How to Install in Termux

$ pkg install git

$ git clone https://github.com/Pegin65/Cursus

$ cd Cursus

$ chmod +x Cursus.sh

$ termux-wake-lock

$ bash Cursus.sh

sudo apt install curl

cd Cursus
